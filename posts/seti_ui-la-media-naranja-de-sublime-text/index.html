Seti_UI la media naranja de Sublime-text

June 12, 2015

Comments

Me involucré en desarrollo allá por el 2010 cuando era un cuasi puberto y tuve mi primer experiencia profesional en el 2011, desde entonces he probado mas de un centenar de IDE's para desarrollo, casi siempre con Eclipse o Netbeans y si bien supe desenvolverme bastante bien con estos entornos nunca deje de sentirme incomodo.

No fue hasta mediados del 2012 que conoci Sublime-text, fue amor a primera vista y desde entonces lo uso hasta para redactar mis recetas de cocina, su simplicidad enamora y ni hablar de su gran variedad de plugins que dia a dia se multiplica.

El dia de hoy lo tengo instalado en mi Debian Jessie x64 con pylinter, flake8, reStructuredText improved, GitGutter y ahora con Seti_UI el cual podran instalar de la siguiente forma:

En muchas distros actuales es posible instalar sublime text desde los repos

  • A tener en cuenta* por las dudas primero verifiquen que este el paquete en los repos
$aptitude search Sublime-text

$sudo apt-get install Sublime-text

Luego de haberlo instalado tienen que tener el *packagecontrol* instalado

Una vez hecho esto solo basta con ejecutar el packagecontrol (ctrl+shift+p) el cual lanzara una caja de texto en la cual pondran install package para finalmente escribir "Seti_UI".

Configuracíon

Preferences -> Settings - User
{
  "theme": "Seti.sublime-theme",
        "color_scheme": "Packages/Seti_UI/Scheme/Seti.tmTheme",
}
