Hello Universe

January 25, 2015

Comments

Finalmente me decidi a abrir un blog despues de mucho pensarlo, se podria decir que casi en la ultima decada he sido colaborador de diferentes sitios web y/o blogs pero ello implicaba adherirse a sus politicas de publicacion incluso en aquellos en los que era su propio administrador.....

En este nuevo ciclo de mi vida me propongo compartir cosas por mis medios, de la forma que a mi me gusta, con muy poco filtro y probablemente algunas faltas ortograficas.

Al principio me habia decidido por usar el sistema blogger de google, supe tener 2 blogs en algun momento que por diversos motivos cayeron en el abismo del olvido, solia pasar horas customizando mis propios themes mientras lidiaba con ese asqueroso y limitado xml, pero no fue hasta hace mucho que por esas cosas del destino me entere que un colega de la comunidad python(Roberto Alsina) habia sido uno de los creadores del proyecto Nikola. el cual ha sido desarollado integramente en Python 3, lenguaje que pretendo dominar en algun momento.

¿Nikola? ¿Que carajo es eso?

Nikola (por Nikola Tesla) es un generador estatico de blogs que ademas permite configurar cada apice de su codigo para customizar a piaccere sus funcionalidades, es bastante facil de instalar pero para poder usarlo hay que deglutir un poco de documentacion para saber donde, como y que tocar para poder empezar a postear.

Al momento de este post la version es 7.3.0 y estas son algunas de las funcionalidades mas destacadas:

  • Pagina principal con los ultimos post
  • RSS Feeds
  • Paginacion de feeds por cada tag
  • Buscador personalizado
  • Historico de post
  • Path personalizados para crear paginas nuevas
  • Templates personalizables
  • Static pages (not part of the blog)
  • Soporte para internalizacion (18 idiomas)
  • Google sitemap generation
  • Custom deployment
  • Formateo de textos livianos (reStructuredText or Markdown)
  • Galeria de imagenes
  • Slideshows para imagenes
  • Client-side cloud tags
  • Edicion en tiempo real

Instalacion

No pretendo hacer un How to sobre como instalarlo ya que para eso tenemos la documentacion oficial de Nikola (en ingles), asi que solo me voy a limitar a mostrar algunos pasos para mostrar cuan facil es tener un sitio funcional en pocos minutos.

Dependencias

Es probable que requieras cumplir con algunas dependencias y para ello es necesario utilizar:

pip install nikola[extras]

De todas formas es posible instalarlas una por una cuando lo requiera.

libxml/libxslt :

Debian y derivados:

sudo apt-get install libxml2-dev libxslt1-dev zlib1g-dev

Red Hat y derivados:

sudo yum install libxslt-devel libxml2-devel zlib-devel

Python.h :

Debian systems:

sudo apt-get install python-dev

Red Hat/RPM-based systems:

sudo yum install python-devel
  • La instalacion fue realizada en un linux mint 17.1

Como pueden ver desde el vamos tenemos la opcion de embeber videos e imagenes, lo unico que necesitamos es ponernos a tono con RST(reStructuredText).

Creamos un sitio en limpio(wizard):
nikola init nombre_de_nuestro_blog

Hint

Es posible generar un sitio de muestra con: nikola init --demo mysite

Para utilizar los siguientes comandos es necesario estar dentro de la carpeta con el nombre que le dimos en el paso anterior.

Para crear un post:
nikola new_post
Editar un post:
Luego de crear un nuevo post, se nos creara una nueva carpeta output la cual se usara para publicar nuestro sitio.
Buildeamos el sitio:
nikola build
Inicializamos el servicio:
nikola serve
Y accedemos desde aqui:
http://127.0.0.1:8000
Pero en lugar de hacer el paso anterior podemos ejecutarlo y editarlo en tiempo real con:
nikola auto -b

tal como se muestra en el video

Para formatear el texto use RST pero para evitar quebraderos de cabeza estoy utilizando un plugin para Sublime Restructured text completion el cual nos permite usarlo con unas cuantas comodidades, por el momento encontre dos, uno es este y fue desarrollado por Martin Gaitan otro developer de la comunidad PYAR.
